Newthings Application Documentation

Newthings uses the NewCode Programming Language. It uses the extension `.nt5`, and provides a lot of features, for example, buttons, text and cameras.


Syntax:

@ Function to add a camera camera(front) @ Function to add buttons button(text, action, position, size) @ Dictionary of actions for buttons actions = { 1: camera.zoomin, 2: camera.zoomout, 3: picture("robot", style=1).visib } @ Function to add a picture picture(name, style)

NewCode Documentation

Newthings uses the NewCode Programming Language. It uses the extension .nt5, and provides a lot of features, for example, buttons, text and cameras.


Camera Functions

@ Show Front Camera
camera(front)

@ Show Back Camera
camera(back)

@ Zoom in and out
camera.zoomin
camera.zoomout

@ Shortcut for adding robot image to camera.
camera.newthings


@ Note: When programming in NewCode, the camera will always be centered.

Buttons

@ Add a button with text "Click Me"
button("Click Me")

@ Add an interactive button with bottom-right position and medium size
button("Click Me", position=bottom-right, action=1, size=medium)
@ See more information about actions below in the "Lists" section.

@ Full syntax
button(name, position, action, size)

@ Default values
name = "Button"
position = center @ If that position already has text or buttons, it will move to a different location.
action = none
size = large

Maps

@ Add map
map()


@ Note that only one map is supported, and it is Newthing's built-in map, "NewMap".

Pictures

@ Add "robot" picture at center position
picture(robot, position=center)

List of available pictures in NewOS 5

robot

Text

@ Add text saying "Hello World!"
text("Hello World!")

@ Add grey text saying "Grey Text" with center-left position
text("Grey Text", color=grey, position=center-left)
@ See more information about actions below in the "Lists" section.

@ Full syntax
button(name, position, action, size)

@ Default values
name = "Text"
color = white
position = center @ If that position already has text or buttons, it will move to a different location.


@ Note that the only text colors that are supported are white and gray.
